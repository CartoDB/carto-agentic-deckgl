# Data Sources

## congestion-zone.json

https://data.ny.gov/Transportation/MTA-Central-Business-District-Geofence-Beginning-J/srxy-5nxn

## nyc-boroughs.geojson

https://gist.githubusercontent.com/ix4/6f44e559b29a72c4c5d130ac13aad317/raw/a7a3a37f2fe054ebc18871b34b023d312668f035/nyc.geojson

## bergen-county.geojson

https://raw.githubusercontent.com/plotly/datasets/master/geojson-counties-fips.json (FIPS: 34003)

## metro-region.geojson

Simplified bounding box for NYC metro area (manually created)

## regional-improvements.geojson

Combined dataset of regional traffic improvement zones, generated by `combine_regions.py` from:
- Manhattan (from nyc-boroughs.geojson)
- Bronx (from nyc-boroughs.geojson)
- Bergen County (from bergen-county.geojson)
- Metro Region (from metro-region.geojson)

## subway-routes.json

https://universeofatoms.com/assets/info_design/nycsubway/data/final/nyctsubwayroutes_100627.json
NYC MTA subway routes with official color codes

## congestion-zone-inverted.geojson

Created by `invert_congestion_zone.py` from congestion-zone.json, creating a polygon covering the USA with the congestion zone as holes
